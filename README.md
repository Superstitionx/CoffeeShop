# CoffeeShop
## 적용된 기술
> Spring MVC 패턴을 이용한 웹 어플리케이션 설계
> JNDI를 통한 데이터베이스 드라이버 및 url 연동
> Rest API와 AJAX를 통해 비동기 기능 구현
> JQuery를 통해 jsp의 기능 구현
> Mybatis를 이용한 Database 설계 및 쿼리문 작성
> Tomcat 서버를 이용한 호스팅 구현
> GoogleMap 같은 API 이용
> AOP를 이용하여 보안기능 구현
> SHA-256 방식을 이용하여 암호화 기능 구현
